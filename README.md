# Timyloom

가천 카카오엔터프라이즈 소프트웨어 아카데미 Timyloom팀 레포지토리 입니다.

<br>

## Purpose

누구나 쉽고 간편하게 프로젝트 및 일정을 효과적으로 관리 할 수 있는 웹 기반의 프로젝트 관리 소프트웨어 개발
